Study Spot

This project is another single page full stack web app that I built because I enjoy working in cafes but often dont know a place near me with free wifi. This site allows user to see a map marked with buisnesses near them with free wifi. It also allows users to add buisness to the map once they are logged in. To make this app I used Ruby on rails and React/Redux and integrated Google's Maps API.
